Instacart Specification Document

This document contains the specifications for Instacart's project.

Features

  • User registration and login
  • Product browsing and searching
  • Shopping cart management
  • Checkout process
  • Order history

Technologies Used

HTML, CSS, JavaScript, Node.js, Express, MongoDB

API Endpoints

  • GET /products
  • POST /users/register
  • POST /users/login
  • POST /cart/add
  • POST /checkout
